# 📖 Overview

The **Smart Meeting Scheduler** is a responsive web application built using **HTML, CSS, and JavaScript** that allows users to create, organize, and manage meetings efficiently.

Instead of manually coordinating schedules, users can quickly schedule meetings, store them locally, edit or delete appointments, and prevent overlapping meeting times.

This project demonstrates front-end development concepts including **DOM manipulation, Local Storage, CRUD operations, responsive UI design, and JavaScript event handling**.

# 🚀 How It Works

### Step 1

Open the application in your browser.

### Step 2

Fill in:

- Meeting Title
- Date
- Time
- Participants
- Location
- Description

### Step 3

Click **Schedule Meeting**

### Step 4

The scheduler checks for time conflicts.

### Step 5

If no conflict exists, the meeting is saved.

### Step 6

Users can

- View meetings
- Edit meetings
- Delete meetings
